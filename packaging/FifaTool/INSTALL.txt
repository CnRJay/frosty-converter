FrostyConvert — FIFA / EA FC tool
================================

Convert compiled FIFA Editor Tool / Mod Manager .fifamod files into
editable .fifaproject files.

Package contents
----------------
  FrostyConvert.FifaGui.exe — simple GUI (double-click to convert)
  fbmod2project.exe         — CLI tool (Windows x64, .NET runtime embedded)
  oodle-data-shared.dll     — Oodle decompression (must stay next to the .exes)
  INSTALL.txt               — this file

Quick start (GUI)
-----------------
1. Unzip this folder anywhere (keep the .exes and .dll in the same folder).

2. Double-click FrostyConvert.FifaGui.exe

3. Browse to a .fifamod (or drag-and-drop), optionally check "Promote legacy
   textures", then click Convert.

4. Open FIFA Editor Tool, load the matching game (e.g. FC26).

5. File → Open Project → recovered.fifaproject

6. Edit assets, then save / export a new mod if desired.

Quick start (CLI)
-----------------
1. Unzip this folder anywhere (keep the .exe and .dll in the same folder).

2. Open a terminal in this folder and run:

     fbmod2project.exe "path\to\mod.fifamod" --inspect
     fbmod2project.exe "path\to\mod.fifamod" -o recovered.fifaproject

3. Open FIFA Editor Tool, load the matching game (e.g. FC26).

4. File → Open Project → recovered.fifaproject

5. Edit assets, then save / export a new mod if desired.

Also works for Frosty .fbmod inventory and offline .fbproject:

     fbmod2project.exe mod.fbmod --inspect
     fbmod2project.exe mod.fbmod -o recovered.fbproject

(CFB/Madden property-grid editing still needs the MMC plugin package.)

Flags
-----
  --inspect              List assets in the mod
  -o, --output PATH      Write .fifaproject or .fbproject
  --json                 JSON report
  --report PATH          Write JSON report to a file
  --extract DIR          Dump raw payloads
  --oodle PATH           Override Oodle DLL
  --inspect-project      Summarize a .fbproject
  -h, --help             Help

Requirements
------------
- Windows x64
- No separate .NET install required (self-contained single-file build)
- FIFA Editor Tool installed to open .fifaproject results
