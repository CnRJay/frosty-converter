Frosty Converter — FIFA / EA FC tool
====================================

Convert compiled FIFA Editor Tool / Mod Manager .fifamod files into
editable .fifaproject files. One exe: GUI when double-clicked, CLI when
you pass arguments.

Package contents
----------------
  Frosty Converter.exe  — GUI + CLI (Windows x64, .NET runtime embedded)
  oodle-data-shared.dll — Oodle decompression (must stay next to the .exe)
  INSTALL.txt           — this file

Windows SmartScreen / antivirus
-------------------------------
This build is not code-signed. Windows or antivirus may warn or quarantine
the .exe as a false positive. That is normal for unsigned open-source tools.
If prompted: More info → Run anyway, or add an exclusion for this folder.
Source is public on GitHub if you prefer to build yourself.

Quick start (GUI)
-----------------
1. Unzip this folder anywhere (keep the .exe and .dll in the same folder).

2. Double-click "Frosty Converter.exe"

3. Browse to a .fifamod (or drag-and-drop), click Convert.
   Leave "promote legacy .dds" unchecked unless this is a crest/UI pack
   with standalone .dds files (faces, scoreboards, gameplay do not need it).

4. Open FIFA Editor Tool, load the matching game (e.g. FC26).

5. File → Open Project → recovered.fifaproject

6. File → Save, then export a NEW .fifamod and test in Mod Manager.
   Skipping Save is the most common cause of “works in editor, broken in game.”

Quick start (CLI)
-----------------
1. Unzip this folder anywhere (keep the .exe and .dll in the same folder).

2. Open a terminal in this folder and run:

     "Frosty Converter.exe" "path\to\mod.fifamod" --inspect
     "Frosty Converter.exe" "path\to\mod.fifamod" -o recovered.fifaproject

3. Open FIFA Editor Tool, load the matching game (e.g. FC26).

4. File → Open Project → recovered.fifaproject → Save → export new mod.

Also works for Frosty .fbmod inventory and offline .fbproject:

     "Frosty Converter.exe" mod.fbmod --inspect
     "Frosty Converter.exe" mod.fbmod -o recovered.fbproject

(CFB/Madden property-grid editing still needs the MMC plugin package.)

Common flags
------------
  --inspect              List assets in the mod
  -o, --output PATH      Write .fifaproject or .fbproject
  --json                 JSON report
  --report PATH          Write JSON report to a file
  --extract DIR          Dump raw payloads
  --oodle PATH           Override Oodle DLL
  --inspect-project      Summarize a .fbproject
  -h, --help             Help

Requirements
------------
- Windows x64
- No separate .NET install required (self-contained single-file build)
- FIFA Editor Tool installed to open .fifaproject results
